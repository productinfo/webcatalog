Web applications like Gmail, Messenger, etc are becoming more and more like desktop applications but running each of these web apps in a separate tab in your browser can be a real pain. WebCatalog lets you turn any web apps into desktop apps (also called site-specific browsers). Now, you can launch your favorite apps quickly in self-contained, distraction-free windows with the ability to switch between accounts and workspaces easily. Each app's data is stored separately, protecting you from cross-website trackers and preserving your privacy.

Catalog - Install apps quickly from our curated catalog.

Custom Apps - Create apps easily from any websites.

AppView - Work just like a real native app.

Workspaces - Switch between multiple accounts/projects.

Menubar - Access your apps anytime and anywhere.

Customization - Customize everything to suite your taste.

Browser Engines - Choose between using Electron, Google Chrome, Mozilla Firefox, Brave, or Vivaldi to power your apps[NOTE].

[NOTE] Many features such as badges, notifications, workspaces and menubar are only available if you choose Electron as browser engine. WebCatalog supports creating apps using other browser options but highly recommends you to use Electron.
