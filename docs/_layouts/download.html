Web applications are becoming more and more like desktop applications but running each of these web apps in a separate tab in your browser can be a real pain. WebCatalog lets you turn any web apps into desktop apps (a.k.a.Â site-specific browsers). Now, you can launch your favorite apps quickly in self-contained, distraction-free windows with many additional features to enhance your workflows. Even better, with sandboxing, it also protects you from cross-website trackers and preserving your privacy.

Catalog - Install apps quickly from our curated catalog.

Custom Apps - Create apps easily from any websites.

AppView - Work just like a real native app.

Workspaces - Switch between multiple accounts/projects.

Notifications - Control notifications with scheduling.

Customization - Make yourself at home!

Browser Engines - Choose between using Electron, Chrome, Edge, Firefox, Brave, or Vivaldi to power your apps[NOTE].

[NOTE] Many features such as badges, notifications, workspaces and menubar are only available if you choose Electron as browser engine. WebCatalog supports creating apps using other browser options but highly recommends you to use Electron.
